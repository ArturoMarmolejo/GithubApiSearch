## Installation

1.- Clone the project on the desired folder:

```sh
git clone git@github.com:ArturoMarmolejo/GithubApiSearch.git
```

2.- Use Node version 14. 

```sh
node 14
```
3.- Install dependencies

```sh
npm install
```


## Running the project:

Run locally:

```sh
npm run dev
```
